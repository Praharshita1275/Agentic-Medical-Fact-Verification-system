MedVerify
Medical Intelligence System

Adversarial AI for
Medical Fact-Checking

MedVerify evaluates health and medical claims against peer-reviewed scientific literature using a multi-agent debate pipeline, hybrid retrieval, and probabilistic uncertainty analysis.

What the Project Does
01
Hybrid RAG Pipeline
Combines a local FAISS vector index with live PubMed NCBI search, pulling the most relevant peer-reviewed evidence for any claim.
02
Multi-Agent Debate
A PRO agent and a CON agent independently argue each side using evidence-backed reasoning, followed by structured rebuttals.
03
Judicial Verdict
An independent judge agent weighs both arguments to produce a final True / False / Uncertain verdict with full reasoning.
04
5-Factor Uncertainty
Confidence is decomposed across five axes — evidence quality, retrieval score, argument balance, model certainty, and conflict level.
05
Source Transparency
Every verdict is backed by retrievable sources. Uncertainty factors are named and explained, not hidden in a black box.
06
Combined Report
A structured report consolidates the verdict, debate arguments, rebuttals, sources, and every metric in one document.
How It Works
STEP 01
Evidence Retrieval
The claim is embedded and used to query a local FAISS index (5 chunks) and the PubMed API (3 articles), returning a unified evidence pool.
STEP 02
Claim Classification
The claim is tagged by domain (nutrition, pharmacology, oncology…) and complexity to route it to the correct reasoning chain.
STEP 03
Adversarial Argument Generation
Two independent agents simultaneously build the strongest possible case for and against the claim using retrieved evidence.
STEP 04
Cross-Examination and Rebuttals
Each agent challenges the other's argument, exposing logical gaps, citation weaknesses, and counter-evidence.
STEP 05
Judicial Verdict
A neutral judge agent weighs the full debate transcript and issues a final verdict with detailed reasoning.
STEP 06
5-Factor Uncertainty Analysis
Confidence, uncertainty, retrieval quality, evidence conflict, and model certainty are computed and returned alongside the verdict.
